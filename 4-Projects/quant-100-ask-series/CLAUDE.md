## 一、项目定位

### 1.1 系列身份

- **系列名**：量化入门100问
- **发布渠道**：小红书小号「11点半收盘吃饭」（`account: xiaohao`）
- **终极目的**：以"知识科普 + 人设可信度"为钩子，**系统性引流到大号**「搞钱最重要」
- **形式**：小红书图文卡片为主，单篇 200–400 字正文 + 配图卡片
- **篇幅上限**：每问的图文卡片数 ≤ 18 张（小红书硬上限）

### 1.2 难度梯度（由浅入深）

| 区间 | 题号 | 主题侧重 | 读者画像 |
|---|---|---|---|
| 入门段 | Q001–Q030 | 量化是什么、常见误解、行业生态、职业路径、基础概念 | 零基础好奇者 |
| 进阶段 | Q031–Q070 | 因子/回测/数据/策略分类、实战踩坑、工具链、风控常识 | 已入门、在自学 |
| 深度段 | Q071–Q100 | 高频/机器学习/组合优化/实盘细节/行业现状与前沿 | 有一定经验者、从业者 |

**铁律**：题号一旦发布**不重排**。topic-pool 里可以先占位，拿到 drafts 再定稿。

## 三、写作基本要求

### 3.1 用词降维

- 尽量**大白话 + 接地气比喻 + 适度 Emoji + 短句排版**
- 少用学院派黑话；出现专业名词**必须在同一段落给出一句人话解释**
- 比喻要贴近生活（买菜、打车、奶茶店、游戏），避免"宇宙飞船"式不切实际的类比

### 3.2 案例与时俱进

- **优先使用 2026 年宏观与最新市场真实行情**作为案例锚点
- 经典历史复盘（2015 股灾、2020 原油负价、2022 雪球敲入、2024 中性策略回撤等）作为补充
- **同一系列内不同题尽量不用重复案例**——每次挑新场景

### 3.3 图文配置

- 需要图的一律配图（策略损益图、回测曲线、流程图、分布直方、相关性热图）
- 图用**相对路径**引用，存在本项目 `assets/QXXX-<问题简称>/` 下
- 图片命名：`QXXX-<问题简称>-<图编号>.png`（如 `Q012-什么是夏普比率-01.png`）

### 3.4 术语标准化（量化版）

| 禁用/弱推 | 统一使用 |
|---|---|
| 看涨期权 / 看跌期权 | 认购期权 / 认沽期权 |
| "多头" "空头"（口语化场合） | "买方/卖方"或"做多/做空" |
| "爆仓" | 保证金不足被强平 / 损失扩大 |
| "杠杆拉满" | 保证金放大效应 |
| 年化收益率（不说口径） | 必须标注：算术年化 / 几何年化 / IRR |

## 四、铁血合规（小红书限流自检，必做）

改写完成后**每题必须逐项扫描**，审核过但被限流等于白写。

1. **投资建议词**：禁用"建议买入 / 推荐操作 / 应该做空" → 改为"该策略适用于…场景"
2. **收益承诺词**：禁用"稳赚不赔 / 保底 / guaranteed profit" → 改为"理论上最大盈利为…"
3. **煽动情绪词**：禁用"必看 / 绝密 / 赶紧上车 / 错过后悔"等标题党
4. **绝对化用语**：禁用"一定 / 绝对 / 100% / 永远" → 改为"通常 / 大概率 / 在…条件下"
5. **金融敏感词**：慎用"杠杆 / 爆仓 / 翻倍" → 用"保证金放大效应 / 损失扩大 / 收益放大"替代
6. **诱导互动**：正文**不加**"点赞收藏 / 关注我"
7. **外部引流**：不放微信号、二维码、外链
8. **敏感话题**：不点评具体政策、监管措施、官员、具体机构好坏
9. **极端行情词**：禁用"暴涨 / 暴跌 / 崩盘 / 恐慌" → 改为"大涨 / 大跌 / 市场波动较大 / 市场情绪偏弱"
10. **个股/具体产品推荐**：全系列禁止点名"建议买 XX"。举例时用"某大盘蓝筹""某商品期货品种"即可，必要时用代码脱敏。

## 五、技术事实核查（每题必做）

写完后在心里跑一遍 checklist：

- [ ] 引用的公式（夏普、索提诺、最大回撤、IC、IR、Beta、Calmar 等）口径正确
- [ ] 回测相关表述未犯未来函数 / 幸存者偏差 / 过拟合
- [ ] 合约乘数、保证金比例、手续费、印花税、滑点等基础要素准确
- [ ] 希腊字母（Delta/Gamma/Vega/Theta）符号与方向正确
- [ ] 波动率（IV/HV/RV）引用合理、区分清楚
- [ ] 关键风险点（无限亏损、流动性风险、基差风险、模型失效）已充分披露
- [ ] 所有数据能追溯到公开来源（交易所、Wind、同花顺、Choice、论文等）

### 5.1 国内文献优先（涉及学术观点时必做）

- 涉及**学术结论、因子研究、A 股有效性、中国市场微观结构**等话题时，**至少引用 1 篇国内文献**（知网、万方、《金融研究》《管理科学学报》《系统工程理论与实践》《经济研究》等）
- **国内文献优先，海外文献作补充**（目标是讲清楚 A 股和国内商品市场，不是美股）
- 引用格式：作者（年份），《论文名》，期刊名，卷(期)：页码
- 搜到的新文献同步补到 `assets/国内文献数据库.md`（没有就新建）

## 六、市场参与者硬约束（⚠️ 与美股的根本差异）

> **绝对禁止**把成熟美股的结构照搬到 A 股 / 国内商品期货市场。

### 6.1 公募基金的边界

- **公募证券投资基金不参与 A 股股票期权市场，也不参与商品期货期权市场**
- 公募基金参与股指期货**仅限对冲且有严格比例限制**
- ❌ 禁写："公募基金买入认沽期权对冲下行风险""公募作为主要买方大量买入认购期权保险"

### 6.2 A 股 / 国内商品期权市场的实际参与者

| 参与者 | 典型角色 |
|---|---|
| 券商自营 | 主要波动率卖方，卖备兑/保险 |
| 私募绝对收益 | 波动率套利、Delta 中性、Gamma Scalping |
| 期货公司风险管理子公司 | 做市商、场外衍生品对冲服务 |
| 保险资金 | 少量买入香草期权，以场外为主 |
| 产业资本 | 买期权做套期保值 |
| 量化私募 | 统计套利、波动率曲面交易 |

### 6.3 其他国内市场硬约束

- A 股 **T+1**、有**涨跌停板**、**融券成本高且受限**
- 股指期货开户门槛（50 万 + 知识测试）、日内交易手续费高
- 商品期货散户参与度高，但**期权以机构为主**
- 科创板 / 创业板 / 北交所有单独适当性门槛

写到涉及"谁在做这笔交易""能不能这么操作"时——**先回到这张表**校验一遍。

## 八、单篇写作结构模板

沿用 xhs-xiaohao 三段式，本系列额外要求：

```
【标题】 Q007｜什么是因子？（<=20 字，带题号）

【开头钩子】（30–80 字）
  └─ 场景 / 反常识 / 人设锚点；不要上来就定义

【正文核心】（150–250 字）
  └─ 1) 一句话人话定义
  └─ 2) 一个接地气比喻
  └─ 3) 一个 2026 年真实案例 / 经典历史案例
  └─ 4) 一个常见误区或踩坑

【金句收束】（30–50 字）
  └─ 让读者"啊对"

【引流钩子】（可选，20–30 字）
  └─ "完整回测代码/数据图表我发在大号了"
```

**每 3–5 题安排一次引流钩子**，不要每题都挂——强行引流会掉数据（参见 xhs-xiaohao §5）。

## 十一、Agent 行为准则（本项目）

**该做**：
- 每题写完先跑一遍第四节（合规 10 条）和第五节（事实核查）自检，把 checklist 结果简短汇报给用户
- 涉及学术结论时先查国内文献（见 §5.1）
- 涉及"谁能做这笔交易"时先对照第六节市场参与者表
- 写完后主动问用户："这题要不要加引流钩子？指向哪篇大号文章？"
- 图表能用 pyecharts/plotly 这类交互图最好，静态图（matplotlib）兜底
- 每篇 publish 后做反哺检查（Content §4.5）

**不该做**：
- 不自动批量写多题（一次一题，写完等用户确认）
- 不编造数据、文献、人物、事件
- 不把美股结构（mutual fund 做备兑等）搬到 A 股
- 不在正文加"关注我/点赞/评论666"
- 不放具体个股/基金代码作为推荐
- 不越界扫描全库——本项目内解决的事不出项目目录
