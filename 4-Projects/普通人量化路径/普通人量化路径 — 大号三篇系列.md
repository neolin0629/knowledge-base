## 各篇内容摘要

### P1：你学的量化，不是在教你赚钱

**钩子**（数据冲击）：2025 年 Jane Street 入门 $300K–$500K，网上 90% 的量化入门教的是这条路——它根本不是在教你。

**核心内容**：
- 专业 Quant 路径 vs 普通人量化路径对比表（5 个维度）
- S1 概率化思维：条件概率（60% → 75%）+ 贝叶斯更新 + 期望值 × 方差 + 凯利半折
- 阶段输出：能写出自己的「条件开仓清单」

**参考资料**：Blitzstein & Hwang《Introduction to Probability》前 6 章

---

### P2：不是 Python 不够，是纪律不够

**钩子**（数据冲击）：随机生成 1000 个策略跑回测，纯靠运气有 50 个显示「显著盈利」。你的前 10 个策略大概率就是这 50 个里的。

**核心内容**：
- 多重比较陷阱：1000 策略 50 个假阳性（p < 0.05）
- Bonferroni 校正（人话版）
- Alpha 自评：线性回归剔除 Beta，看截距是否显著
- 交易日记标准化：进出场理由 + 止损位 + 事后归因

**待确认**：López de Prado《Advances in Financial Machine Learning》Chapter 8 具体论点（多重比较 / combinatorial purged cross-validation）——动笔前必须读完。

---

### P3：你不需要学这些（工具已经民主化，判断力没有）

**钩子**（金句）：「Tools have democratized. Conviction hasn't.」任何人都能 pip install vectorbt，优势在判断力和纪律，不在工具。

**核心内容**：
- 最小工具链对比表（Python + Tushare/AkShare + vectorbt + statsmodels）
- 完整闭环结构图（数据 → 信号 → 回测 → 统计验证 → 模拟盘 → 实盘）
- 明确不学清单（Black-Scholes / C++ / QuantLib / Shreve / NautilusTrader）——每条给原因，不是「用不到」，是「跟你的护城河无关」

**结尾**：这条路不性感，但能让账户活下来。
