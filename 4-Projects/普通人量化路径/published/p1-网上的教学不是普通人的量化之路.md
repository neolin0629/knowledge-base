## 正文

2025 年，顶级量化公司（Jane Street / Citadel / HRT）入门年薪是 $300K–$500K。

搜索「普通人量化入门」，排名靠前的内容，90% 在教你推导 Black-Scholes、啃随机微积分、搭 C++ 低延迟系统。那条路的终点是机构 Quant 职位。和你的账户，没什么关系。

**两条路径，从起点到终点都不一样**

|       | 专业 Quant 路径           | 普通人量化路径           |
| ----- | --------------------- | ----------------- |
| 数学门槛  | 随机微积分、伊藤引理            | 条件概率、线性回归         |
| 目标    | 机构年薪 $300K+           | 账户从赌博变正期望系统       |
| 核心护城河 | PhD + 低延迟基础设施         | 概率思维 + 纪律         |
| 典型工具  | C++ / Rust / QuantLib | Python + 免费数据 API |
| 学习周期  | 18 个月啃 5 本专业教材        | 12 个月建最小闭环        |

2025 年 A 股，81.1% 的散户全年亏损（东方财富，2025.12）。不是因为没学 Black-Scholes，是因为没有一套有边界的系统——不清楚自己在什么条件下该进场、什么时候就是不动。

**S1：从「感觉」到「条件」（2–3 个月）**

第一步不是装 Python，是把「感觉它会涨」改写成「在这个条件下，涨的概率是 75%」。

举个具体例子：某资产每天上涨的基础概率是 60%——这是背景噪音，没有指导意义。但在成交量高于 20 日均值的交易日，上涨概率变成 75%——这才是值得入场的信号。其他日子，看都不看。

S1 阶段只有一个输出：写出「我只在 XX 和 XX 条件同时成立时开仓」，并能估算这个条件历史上出现的频率。能写出来，S1 就过了。

**仓位管理：凯利公式，只用一半**

凯利公式算出理论最优仓位比例，但它有一个前提：你的胜率估算是准的。你永远不准。

实战规则：算出凯利值，打对折。留一倍误差缓冲，才能扛过连亏 8 次不爆仓。

——

S1 是入门关。真正把大多数人筛掉的，是 S2——你跑出来的前 10 个策略，大概率不是本事，是统计噪声。下一篇讲这个。
