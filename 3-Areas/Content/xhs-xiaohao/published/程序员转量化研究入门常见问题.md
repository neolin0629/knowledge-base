# 📌 标题：程序员转行量化？先搞懂这几个问题🔥

作为程序员，转行量化真的那么香吗？最近几天被问到了很多问题。今天就用大白话带大家盘点程序员转型量化的【5个灵魂拷问】，看完再决定要不要入坑👇以下所有回答有个大的前提假设：你是一位正常职场中的程序员。

## 📝 正文大纲

**1⃣️ 程序员转行量化怎么起步？选哪条路？**

先说个大概情况，AI coding 能力的大爆发，对于纯程序员来说很不友好。我自己所在的团队，现在也基本不招实习生和纯程序员了。

之前最稳妥的路径是通过 QD 进入量化私募，通过写系统、清洗数据，近距离学习研究，大概2年逐步尝试独立写策略。这条道路已经基本不通了。

现在比较推荐的路径是：不要裸辞，业余时间自己尝试写代码跑策略，得到实打实的研究成果，然后一边自己做投资积累经验和业绩，一般去面试研究岗。

💡 **自己做 vs 进公司怎么选？** 强烈建议先争取进大平台！大平台不仅给你发稳定工资，还有你个人根本搞不到的昂贵数据、庞大算力和成熟的研究框架。稳定的现金流下，你的交易都会变得更加从容。

**2⃣️ 全职自己做，需要多少本金？**
想要全职，你需要算两笔账：
- 🍕 **生存成本**：至少准备 1-2 年的纯生活费（和你所在的城市有关）, 假设18万。
- 💰 **策略资金**：这取决于你做什么品种：（加密货币先不谈）
	- 如果做A股量化选股，考虑到分散持仓（至少20只）和各种交易权限的开通，起步资金大概需要30-50万。
	- 如果做商品期货CTA，利用保证金杠杆，10-30万可以起步，但如果要做多品种组合分散风险，50万会更从容。
	- 如果做期权策略，看你的策略类型，卖方策略需要的保证金较多，20万比较合理；买方策略可以小资金，但波动大、容错低。

举个例子：假设你有100万策略资金，年化做到25%（在自营这个层面，不算特别高但需要持续稳定），一年收益25万，扣掉生活成本18万，剩7万。坦白讲这不算宽裕，你的容错空间很小。

所以我个人的建议是：**策略资金至少100万 + 生活备用金36万，合计至少准备130-150万，并且你的策略已经有至少6-12个月的实盘验证**，再考虑全职。

**3⃣️ 现在量化还好赚钱吗？**

实话实说：**没前几年好赚了。** 现在量化圈高度机构化，都在卷算力、卷数据，“Alpha（超额收益）”越来越难找且衰减极快，闭眼捡钱的时代早过去了。

程序员的优势在于**数据处理能力、工程落地能力**。现在确实有各种 AI Agent 工具，但那个是未来，而不是现在。纯黑箱处理的策略，亏钱的时候你都不知道为什么。

所以自己一步步搭轮子，把自己的交易理念（或者是别人的）抽象成程序语言，在最终实现自动化，才是程序员应该专注的道路。

**4⃣️ 技术大佬转量化，还需要补什么课？**
基本上，要对下面的内容有所了解：
- 🧮 **数学与统计**：概率论、线性代数、时间序列分析（这块是核心中的核心）。
- 📉 **金融业务**：市场微观结构（品种的交易规则和特性，订单簿？）、金融衍生品基础。
- 🤖 **多因子/机器学习**：你可以不用，但是不能不懂，而这恰好是程序员的统治区！
