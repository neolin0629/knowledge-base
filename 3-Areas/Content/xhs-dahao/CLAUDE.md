## 1. 账号定位

**专业金融 × 数据驱动 × 未来商业化**

- 受众：财经爱好者、量化从业者、潜在付费用户
- 调性：干货、权威感、不煽情、不套话
- 商业化方向：知识付费、社群、课程（未来 6-12 个月）

---

## 2. 内容规则（铁则）

### 2.1 溯源铁则（严格执行）

**所有大号文章必须有实质 source_notes 或 source_raw**。不允许空溯源发布。

如果一个选题找不到笔记或 Raw 支撑，要么：
- 回头补笔记
- 降级到 xiaohao 发布
- 放弃

### 2.2 category 允许列表

`量化` / `宏观` / `商品` / `AI投资` / `方法论` / `市场结构`

**不允许**：`生活随笔` / `心情` / `日常` （这些留给 xiaohao）

### 2.3 数据规范

- 任何具体数字/日期/业绩必须标注来源
- 引用他人观点必须有出处（`source_raw` 填 Raw 路径或 URL）
- 不得编造或猜测数据。不确定就说"需查 Raw"

### 2.4 配图规范

- 大号必须有**数据图**或**结构图**（Python + Canva 为主）
- 封面深色底 + 大字标题 + 简洁
- 不用 emoji 作为主要视觉元素

---

## 3. 写作结构（大号图文 500-700 字）

```
钩子（50-80 字）
  └─ 反常识数据 / 冲突观点 / 具体场景

核心结论（80-120 字）
  └─ 1-2 句说清楚，不绕弯

3-4 个论证（每点 80-120 字）
  └─ 每点必须有数据或案例支撑

延伸思考或反问（50-80 字）
  └─ 引导评论

（可选）引流到公众号长文版
```

---

## 4. 钩子类型偏好

| hook_type | 使用频率 | 说明 |
|---|---|---|
| 数据冲击 | 高 | "只有 3% 的量化策略能跑赢沪深 300" |
| 反常识 | 高 | "越勤奋的交易者，收益率越低" |
| 金句 | 中 | 精炼总结一个道理 |
| 故事 | 低 | 大号用故事要 careful，避免太像小号 |
| 人设 | 不用 | 留给 xiaohao |

## 5. 发布节奏目标

- 频率：**周 3-4 篇**（稳定）

- 周末避开上午（金融读者不活跃）

## 7. 选题来源优先级

1. `3-Areas/Investing/Quant/Journal/` 的反哺洞察（最优质，数据支撑最足）
2. `3-Areas/Investing/` 下各主题的 active 观点页

4. `4-Projects/` 系列文章的衍生
5. 热点事件但有能力独特角度的
6. `8-Raw` 中比较有价值且适合的文章

**不采纳**：纯新闻复读、情绪发泄、无数据观点

## 8. 大号特有的 Claude 行为

### 写作时
- 自动查相关 `8-Raw/research-reports/` 有无支撑数据
- 自动检查声明和 Areas 笔记是否一致（如果不一致，提醒我）
- 生成 3 个标题：一个数据导向、一个反常识、一个提问式
- 预估字数和图表数
- 联网搜索国内学术文献（知网、万方、金融研究、管理科学学报等），看是否有观点补充，并在文末写上引用
- 引用格式须包含作者、年份、期刊/出版社、页码（没有的项不填）

### 发布前检查
- source_notes 或 source_raw 非空
- category 合法
- 数据有出处
- 没有"本文不构成投资建议"这种废话套话（我会自己决定要不要加）
- 图表建议的数据可追溯到 Raw 或笔记
