# 1. 伯努利模型（Bernoulli Model）

伯努利模型描述的是一个**二值随机试验**（如抛硬币、点击广告、生病/健康等），其随机变量 \( X \) 只有两种可能：
- $X = 1$（“成功”），概率为 $p$；
- $X = 0$（“失败”），概率为 $1-p$。

其概率质量函数（PMF）为：$P (X = x) = p^x (1-p)^{1-x}, \quad x \in \{0,1\}$]

---
# [[极大似然估计 | 2. 极大似然估计（MLE）]]
## **(1) 观测频率的含义**

假设我们观测到 $N$ 次独立伯努利试验，其中 $k$ 次成功 $X=1$，$N-k$ 次失败 $X=0$。  
**似然函数**（观测数据出现的概率）为：$$L (p | X) = \prod_{i=1}^N p^{x_i} (1-p)^{1-x_i} = p^k (1-p)^{N-k}$$
**对数似然函数**：
$$\ln L (p | X) = k \ln p + (N-k) \ln (1-p)$$

**求导并最大化**：
$$\frac{\partial \ln L}{\partial p} = \frac{k}{p} - \frac{N-k}{1-p} = 0 \quad \Rightarrow \quad \hat{p}_{\text{MLE}} = \frac{k}{N}$$

**结论**：  
MLE 估计的 $\hat{p}_{\text{MLE}}$ 就是**观测到的成功频率**（成功次数 / 总试验次数）。

**“概率最大”的含义**：  
在 $p = \frac{k}{N}$ 时，当前观测数据出现的可能性（似然）最大。

---
# **[[贝叶斯算法 | 3. 贝叶斯估计（Bayesian Estimation）]]**
## **(1) 模型设定**

贝叶斯方法引入**先验分布**（prior），假设 $p$  服从 **Beta 分布**（共轭先验）：
$$P \sim \text{Beta}(\alpha, \beta)$$

其中 ($\alpha, \beta > 0$) 是超参数，控制先验信念（如 $\alpha = \beta = 1$ 时为均匀分布）。

## **(2) 后验分布**

观测数据后，后验分布仍为 Beta 分布：
$$P | X \sim \text{Beta}(\alpha + k, \beta + N - k)$$

**后验均值（贝叶斯估计）**：
$$\hat{p}_{\text{Bayes}} = \mathbb{E}[p | X] = \frac{\alpha + k}{\alpha + \beta + N}$$

## **(3) 与 MLE 的区别**

- **MLE**：仅依赖数据 $\hat{p} = \frac{k}{N}$，可能过拟合小样本。
- **贝叶斯估计**：结合先验（如 $\alpha = \beta = 2$ 时， $\hat{p} = \frac{k+2}{N+4}$），避免极端估计。

---
# **4. 贝叶斯估计的模型、损失函数与求解**
## **(1) 模型**
- **生成模型**：贝叶斯方法假设 $p$ 是随机变量，其分布反映不确定性。
- **区别于条件概率**：MLE 是频率学派方法，视 $p$ 为固定参数；贝叶斯则通过后验分布描述 $p$ 的置信度。

## **(2) 损失函数**
- **平方损失**：最小化 $\mathbb{E}[(p - \hat{p})^2]$ 时，最优估计是后验均值 $\hat{p} = \mathbb{E}[p | X]$。
- **0-1 损失**：若选择后验众数（MAP），即 $\hat{p} = \arg \max P (p | X)$ 。

## **(3) 求解最优解**
1. **选择先验**（如 $\text{Beta}(\alpha, \beta)$)。
2. **计算后验分布**：
$$P (p | X) \propto P (X | p) P (p) = p^{k+\alpha-1} (1-p)^{N-k+\beta-1}$$

3. **求后验均值或众数**：
   - 后验均值：$\hat{p} = \frac{\alpha + k}{\alpha + \beta + N}$。
   - 后验众数（MAP）：$\hat{p} = \frac{\alpha + k - 1}{\alpha + \beta + N - 2}$（当 $\alpha, \beta > 1$）。

---
# **5. 对比总结**

| 方法      | 估计公式                                              | 特点              | 适用场景       |
| ------- | ------------------------------------------------- | --------------- | ---------- |
| **MLE** | $\hat{p} = \frac{k}{N}$                           | 完全依赖数据，可能过拟合小样本 | 大数据、无先验信息时 |
| **贝叶斯** | $\hat{p} = \frac{\alpha + k}{\alpha + \beta + N}$ | 结合先验，平滑估计       | 小样本、需引入先验时 |

---
# **6. 实例**

- **数据**：抛硬币 10 次，7 次正面（ $k=7$ , $N=10$ ）。
  - **MLE**：$\hat{p} = 0.7$。
  - **贝叶斯**（先验 $\alpha = \beta = 2$）：$\hat{p} = \frac{7+2}{10+4} \approx 0.643$。

贝叶斯估计在少量数据时更稳健，而 MLE 在大数据下更准确。
