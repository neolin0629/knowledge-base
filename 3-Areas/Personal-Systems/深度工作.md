# 原理

**最小阻力原则（The Principle of Least Resistar）**

在工作环境下，若行为对于工作产出的影响没有得到明确的反馈意见，我们倾向于采用当下最简单易行的行为。

**忙碌代表生产能力（Busyness as Proxy for Productivity）**

在工作中，当生产能力和价值没有明确的指标时，很多知识工作者都会采用工业时代关于生产能力的指标，以可视化的方式完成很多事情。

**游手好闲是万恶之源**

加拉格尔是这样描述这种神经学陷阱的：“专注报道注意力相关话题已经5年多了，我了解了一些令人不快的真理，其中之一就是 `游手好闲是万恶之源` 。当你失去关注点时，头脑不会想着生活中的好事，而会纠缠于生活中可能出现的问题。”

# 深度工作

**几种深度工作哲学**

1. 禁欲主义哲学（Monastic Philosophy）
2. 双峰哲学（Bimodel Philosophy）将个人时间分成深度和浮浅两块。最短的深度时间在一天以上
3. 节奏哲学（Rhythmic Philosophy）用固定的惯例来进行深度工作
4. 新闻记者哲学（Journalistic Philosophy）灵活地，随时进行深度工作

**习惯化**

-  你将在何处工作，工作时间多长
    - 设定一个具体的时间框架，将这个过程保持为一个离散型的挑战
- 工作开始之后你就如何继续工作
    - 制定规则和程序，比如约定不准使用手机，或者每20分钟的文字产出量，以保持持续专注
- 支持自己的工作
    - 养成习惯来支持大脑的深度工作，比如喝一杯上好的咖啡就开始工作，或者是插入轻度活动，把环境整理的井井有条等

**大手笔**

类似于比尔盖茨的“思考周”，做出大的环境的改变，来促使自己进入深度工作。可以是金钱投入，或者是精力投入来改变环境。

**协作**

追寻创新的工作，协作也能产出好的效果。但是要避免纷扰心神之事。

**4DX：**

- 将关注点放到极端重要的事情上
- 抓住引领性目标
    - 比如专注于重要目标上的深度工作状态时间
- 准备一个醒目的计分板
    - 在公开的地方记录、跟踪引领性指标
- 定期问责

# 拥抱无聊

找到最容易使你分心的事物，然后计划处理或者接触该事情的时间段。比如“定期使用网络”。

**有成果的冥想**：在身体劳作而心智空闲的时候（比如走路、慢跑、开车、淋浴），将注意力集中到一个定义明确的专业难题上。

- 小心分心和原地打转般的思考
- 组织你的深度思考

**记住一副牌**

- 按次序记住选中的房间中的物件。
- 把52张牌与一个难忘的人或东西相对应。
- 在脑海中穿过家，没遭遇一件物品，就看一张牌，然后想象这张牌对应的难忘的人或东西在物品旁边做一件难忘的事。

# 远离社交媒体

**工具选择的手艺人方法**：明确在你的职业和个人生活中决定成功与幸福的核心因素。只有一种工具对这些因素的实际益处大于实际害处时才选择这种工具。

应用目标管理的方法，制定自己的宏观目标，在选择工具的时候，分析工具所带来的积极和消极影响，只有当积极影响大于消极影响时，才使用。

**规划你的娱乐时间，像你所希望的生活那样去娱乐。**

# 抛弃浮浅工作

深度工作要求你尊重自己的时间。

提前决定你一天的每一分钟要做什么工作。或者在一天里时常拿出一点时间来询问自己“在今天剩下的时间里，我做什么最有益处”。

**定量分析每一项活动的深度**

先问一个简单问题来评估任务：“要让一个刚毕业、还没有在该领域接受特别训练的大学生完成这项工作需要多久？”

**控制浮浅工作在预算范围内**，一般是占你工作时间的30-50%。

**下午5点半之前结束工作**

- 采用固定日程的方法，在做选择的时候有所侧重
- 谨慎地思考个人的组织习惯，在有限的时间里产出高价值

Tips：明确的拒绝浮浅工作，但拒绝的理由模糊不清。避免给请求人提供机会，来否定你具体的拒绝理由。

**电子邮件**

- 让写邮件的人做更多的事情
    - 设置发件人过滤器
- 收发电子邮件的时候做更多工作
    - 流程导向回应。思考：这条信息指向哪一个项目，成功完成项目的最有效流程（就产生的信息数量而言）是什么？
- 不要回复下面这些邮件（存在特例）
    - 邮件相当含混或者存在其他问题，很难给一个合理的答复
    - 你对这个问题或者提议不感兴趣
    - 如果你回复了，不会有好结果；如果你不回复，也不会有坏结果
