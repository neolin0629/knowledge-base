人们以为做决定需要的是勇气。但有一类决策，越敢做决定的人，输得越惨。

你有在高压下做过一个后来后悔的重大决定吗？其实，那一刻真正犯的错不是选错了，是你不应该在那个时刻做任何选择。

**在冷静的时候，预先承诺好不同情境下的行动规则，让自己在高压、高情绪的时候不需要临场决策。**

## 决策困境

之前看安妮杜克的《对赌》，书里介绍了一套在人生的重大时刻“如何做决策”的方法论，总结下来是下面三点：

1. 小事，没影响，随便选
2. 大事，错了会出局，认真衡量备好退路
3. 难事，估胜算，听专业意见，理性决定

其实这个三分法是有一个天然裂隙的，**「小/大/难」不是同一个维度**。一件事可以是「大事 + 难事」，也可以是「小事 + 难事」。认知的混淆会导致决策的困难。
* 「小」和「大」是**赌注大小**（stakes）
* 「难」是**认知难度**（uncertainty）

那我们应该如何思考？还记得**凯利公式**吗？
$$
f^* = \frac{bp - q}{b} = p - \frac{q}{b}
$$

其中，$f∗$ 最优投注比例，$b$ 为净赔率（通常指扣除本金后的收益倍数），$p$ 为获胜概率，$q=1−p$为失败概率。

一个用来做投资策略的公式，完美的解决了这个“决策困境”。

## 概念还原

「小事/大事」最初是民间口语，不是分析工具。它从来就没有严格定义。当你说「大事：错了会出局」，你其实在描述的是**不可逆性**（irreversibility）+ **赌注上限**（stake size）。当你说「难事：估胜算」，你在描述的是**不确定性**（uncertainty）。

你真正想说的是两件事：

> 1. **赌注有多大**（输了有多惨，能不能恢复）
> 2. **胜率能不能估**（你有没有信息优势）

而这两个维度的组合，才是真正的决策矩阵：

|           | 赌注小（输得起）    | 赌注大（输不起）        |
| --------- | ----------- | --------------- |
| **胜率可估**  | 随便选或用 Kelly | 精确计算 Kelly，保守下注 |
| **胜率不可估** | 随便选，成本低     | **不下注。等信息。**    |
## 预承诺系统

哈耶克说，人类理性有一种倾向：以为可以设计一套规则，让复杂的自发秩序听话。但现实是：**做决定所需要的信息是分散的、局部的、往往是隐性的，无法被事先整合进一套规则里。**

「决策系统」这个词，默默预设了一件事：**决策的场景是可以事先分类的。** 但真实生活里，「这到底算大事还是难事」本身就是一个决策，而且常常是最难的那个。

所以我们真正想构建的不是「系统」，是**预承诺机制**（pre-commitment）。

这是两件完全不同的事：

| 「决策系统」的幻觉     | 「预承诺机制」的现实          |
| ------------- | ------------------- |
| 输入情境 → 系统给你答案 | 事先决定好：当 X 发生时，我一定 Y |
| 需要系统在现场运行     | 规则已经内化，执行时不再思考      |
| 假设所有情境可枚举     | 只承诺你能提前识别的类别        |
| 替代判断          | 减少不必要的判断            |

凯利公式是最典型的预承诺：你不是在每次下注前实时算一遍，你是在开始前就承诺「我永远不赌超过 $f*$ 的仓位」。这条规则的价值，恰恰在于**让你在贪婪和恐惧的时候不需要重新做决定**。

## 被忽略的第四象限

当一件事不可以估算出胜率且赌注很大时，我们该如何做决定？比如，没有做过尽调的创业或者转行，在情绪最激烈时候的 all in。

凯利公式已经给了我们答案，当 $f*$ 不可计算时，意味着你不应该投注。

**拒绝决定也是一种决定，弃权不是情绪，还是数学论证的最优解。**

所以，当你身处第四象限时，**停止做决定！开始等待！** 等待胜率和赌注大小更加明晰的时候，再做定夺。

## 为什么越有勇气越危险

行为金融学提供了机制解释：投资者在高情绪状态下，存在系统性的**过度自信偏差**（overconfidence bias）——高估自己对胜率的判断能力，同时低估不确定性的真实范围。

赵振全、李晓周（吉林大学数量经济研究中心）在综述中指出，投资者常表现出"过分自信"——高估判断力，低估环境不确定性，这在市场波动剧烈时尤为显著。

**感觉最确定的时刻，往往是最不该下注的时刻。**

「越有勇气」的人，不是胜率更高，而是对胜率的高估更严重。他们下注的不是经过计算的 $f^*$，而是情绪驱动的 $f^* = 1$（全仓）。落在第四象限的决定里，看起来最果断的那些人，往往亏得最惨。

## 附：凯利公式举例

凯利公式是 John L. Kelly 在 1956 年提出，最初用于信息论领域，后来被应用于投资领域。

**硬币实验**

抛硬币，当正面向上时，资金翻一倍，反之则损失投注资金。假设在无限次游戏的情况下，正面向上的概率是 60%。如果在有限资金的情况下，进行有限次投注，那么每次的最优投注是多少钱？  
  
根据公式，b = 100% / 100% = 1, p = 60%, q = 40%, 可以计算出 f = 20%。

则最优投注比例为每次投注 20% 。

**变式**

凯利公式有各种变式，在投资中用的较多的是分数版和收益优化版。

**分数版** 的凯利公式是在原凯利公式计算结果的基础上，乘以一个分数，比如 1/4 凯利，就是 20%✖️1/4，主要是为了增加容错率。

**收益优化版** ，因为凯利公式是二元假设，即输的时候损失所有，而真实的投资中，并不是这样的情况，所以才有了收益优化版。
$$
f* = \frac{p}{a} - \frac{q}{b}
$$
 其中，$b$ 是获胜时的利润，$a$ 是失败时的损失。
